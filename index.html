Capabilities

Built different,
by design

01
RAG-Powered Knowledge

Every answer is grounded in a verified Supabase vector knowledge base — real career paths, actual certifications, and verified salary ranges. Not guesswork.

02
Specialist Routing

Two dedicated agents — one for tech streams, one for non-tech backgrounds. Each has a distinct system prompt, knowledge base, and counselling methodology.

Persistent Memory

A 10-message context window keeps conversations coherent. Your counsellor remembers what you said earlier and never repeats intake questions.
